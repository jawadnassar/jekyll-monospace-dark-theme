# jekyll-monospace-dark-theme
Inspired by textonly small site [0xe9](https://0xe9.net), this is a [Jekyll](https://jekyllrb.com) theme built with [Bootstrap](https://getbootstrap.com) and designed for speed, clarity, and simplicity. 

Ideal for developers who want a fast, lightweight blog without unnecessary complexity.
